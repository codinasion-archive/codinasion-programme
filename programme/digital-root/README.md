## Write a programme for Digital Root of a number

`Digital Root` is the single number obtained by adding the number successively.

```txt
347 = 3 + 4 + 7 = 14
14 = 1 + 4 = 5
```

Thus, 5 is a single digit number, which is the digital root/ seed number of 347.

```txt
Input  : 347
Output : 5
```
